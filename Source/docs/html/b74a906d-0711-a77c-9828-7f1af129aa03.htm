Remarks

Only IRemoteHubAdapterT of byte array (byte[]) is supported as adapter.

Though only byte array is only the acceptable type parameter of adapter, all adapter which encoded private messages as byte array could be supported due to no private message will be decoded in RemoteHub Switch.

For example, a stream, which is connected to a instance of IRemoteHubAdapterT of string in another endpoint, can be handled by a instance of IRemoteHubAdapterT of byte array (byte[]) for linked to this RemoteHub Switch instance.
