Using Redis database

Remote Hub can use Redis database as the medium to connect clients.

Use Redis database

Redis Adapter is the adapter designed by using publish-subscribe service on a Redis database as the medium to connect clients.

All clients connected to the same Redis database can discover each other and transport message among them.

In the solution above, the 3 Redis clients are connected to the same Redis database.

Redis database is specified by configuration string, which need to be passed as argument while constructing the Redis Adapter and Redis Client.

Security

Due to no data encrypting or authentication is handled by Remote Hub, while using Redis database in an open area, like Internet, user need to take care about network security.

Exposing Redis database on public network is strongly NOT RECOMMENDED.

Work with streams
With attached Redis Adapter, Switch instance can also be used to dealing with the messages from and to the clients on the same Redis database.See Using streams for details.
