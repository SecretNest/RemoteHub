Interfaces of Remote Hub

Interfaces of Remote Hub, SecretNest.IRemoteHub, is a library contains interfaces, delegates and shared classes of Remote Hub.

Use this library

This library defines most functions used in Remote Hub adapters and clients.

A user of Remote Hub does not need to reference this library directly. This is handled while your project references other libraries.

  • SecretNest.IRemoteHub.Redis inherits interfaces in this library and adds methods which is used with Redis database only.

  • SecretNest.IRemoteHub.Stream inherits interfaces in this library and adds methods which is used with streams only.

  • SecretNest.RemoteHub.SwitchDirect implements the adapter and client interface directly.

  • SecretNest.RemoteHub.Switch accepts the adapters based on the interface defined in this library.
