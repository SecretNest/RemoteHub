Demo: Chat room

Here is a demo of Remote Hub works with Remote Agency, provided a client server mode of chat room.

Code

Demo code is written in C#. Source code can be found in the folder DemoWithRemoteAgency.

Projects

There are 5 projects in the same solution file.

  • ChatClientOnRedis: The client side of the chat room based on Redis database. Use it with ChatServerOnRedis.

  • ChatServerOnRedis: The server side of the chat room based on Redis database. Use it with ChatClientOnRedis.

  • ChatClientOnSslStream: The client side of the chat room based on streams. Use it with ChatServerOnSslStream.

  • ChatServerOnSslStream: The server side of the chat room based on streams. Use it with ChatClientOnSslStream.

  • SharedInterface: The interface of the ChatServer, name IChatServer. This project is referenced by all other 4 projects listed above.

Demo

Server side includes an instance of ChatServer, which contains a method and an event. When the method received a message, the event will be raised to broadcast it to all handlers.

Due to the stateful management, all service object with event support need to be disposed required by Remote Agency. When a disconnecting is discovered on the server side, method OnProxiesOfSiteDisposed is called to removing the handler related.

Client side should be started after the server side application. A user name need to be entered, then the client side is prepared and connect to the server. Each text entered will be broadcast to all clients.

Running the demo based on Redis need a Redis instance running in localhost, or on another server with changing the connectingString on both server and client side code.

Stream demo is based on Stream Adapter with SslStream. All certificates used is creating on the fly.
