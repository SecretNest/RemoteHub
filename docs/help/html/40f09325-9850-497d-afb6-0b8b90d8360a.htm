Redis Client

Redis Client, SecretNest.RemoteHub.Redis, is a library contains the Remote Hub client based on Redis database.

Use this library

This library defines the Remote Hub client for connecting to Redis database, named as RemoteHubOverRedisT, which implemented IRemoteHubOverRedisT.

Publish-subscribe service is used on the Redis database specified while constructing an instance of this client. No data in this database will be queried or changed.

In this client, a Redis Adapter will be created to handle the operations on Redis database.

Type parameter

Type parameter of Redis Client will be passed to Redis Adapter for sending and receiving user messages.

Currently, only byte array (byte[] in C#.Net / Byte() in VB.Net) and string are acceptable. Passing other type while creating while cause an exception thrown.

Using different type parameters or encoders on clients created on the same Redis database is acceptable. For example, if one side is set as string with one other is using byte array, the text sent from the 1st one will be received by the 2nd as the encoded output. The encoding format is specified by Redis.

The type is applied on the user messages only. System messages, like those for client discovering, will not be affected.

Callback

A instance of OnMessageReceivedCallbackT need to be prepared for receiving user messages. Type of the delegate is the same as the type parameter passed into the Redis Client.

The callback instance need to be passed as an argument of the constructor of Redis Client.
